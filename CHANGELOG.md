# 更新日志 / Changelog

本项目遵循 [语义化版本 SemVer](https://semver.org/lang/zh-CN/)。
格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

## [1.4.0] - 2026-07-02

### 新增
- **等高线 / 地形**图案支持 **SVG 矢量导出**（此前仅反应-扩散可用）。
  - 等高线以**描边路径**输出（非填充），主线 / 普通线自动分组，颜色与加粗跟随当前参数。
  - 复用「SVG 简化(RDP 减面)」与「SVG 平滑(贝塞尔)」两个滑块。

### 修复 / 规避的坑
- 不再对场加背景 padding，避免在瓦片边缘画出**假边框线**（padding 只适用于填充闭合，不适用于描边）。
- 复用**双向链接**算法，修复线条碎裂 / 断线。
- 修正 marching squares 位掩码 case 11 / 13 取边错误。
- 新增 `openBezierD`，开放折线走开放贝塞尔、闭合环走闭合贝塞尔，不再强行 `Z`。

## [1.3.0]

### 新增
- 反应-扩散 SVG 矢量导出的路径优化：RDP 减面 + 贝塞尔平滑滑块。

## [1.2.0]

### 新增
- 懒人调节：线条密度 / 线条粗细两个滑杆。
- 更宽的密度调节范围。
- 预览清晰度提升（DPR + 平滑）。
- 上传孔位保留原始宽高比（居中留白，不拉伸）。
- 反相蒙版可切换生长的内外区域。
